🛒 发卡平台对接
方式一：自动发卡（推荐）
1. 部署 独角数卡 发卡系统
2. 在独角数卡后台 → 商品管理 → 新建商品 → 商品类型选「自动发卡」
3. 在下方"生成卡密"批量生成卡密，然后点击"导出CSV"
4. 将导出的卡密列表粘贴到独角数卡商品的库存中
5. 配置微信/支付宝支付参数，上架商品即可
6. 用户支付后独角数卡自动从库存取一张卡密发给买家
批量生成 + 导出
1. 在下方"生成卡密"区域设置额度和数量
2. 点击"生成"按钮
3. 点击"导出CSV"下载所有卡密
4. 将CSV导入到任何发卡平台的库存中
Webhook 自动生成卡密
当用户在发卡平台支付成功后，发卡平台调用你的 Webhook 接口自动生成卡密。
